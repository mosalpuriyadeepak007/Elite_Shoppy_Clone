## 📖 About The Project

**Elite Shoppy Clone** is a front-end e-commerce website built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

The project recreates the look and feel of a modern online shopping platform, with a focus on clean UI design, responsive layouts, smooth animations, reusable components, and engaging interactions.

It's a great showcase of core front-end skills: responsive web design, CSS animations, JavaScript DOM manipulation, image sliders, product filtering, dropdown navigation, and shopping cart UI implementation.


**2. Navigate to the project**
